# Práctica UNED "Sistemas distribuídos"

Mi práctica de la asignatura de la UNED "Sistemas Distribuidos" del curso 2018-2019.

Contenidos del repositorio:

```shell
.
├── basededatos     # Código fuente del servicio "Base de datos"
├── basededatos.bat # Iniciador del servicio "Base de datos" (WIN)
├── basededatos.sh  # Iniciador del servicio "Base de datos" (*nix)
├── common          # Código fuente compartido
├── dist            # Binarios con los servicios precompilados
├── doc             # Documentación de la práctica
│   ├── assets      # Imágenes auxiliares usadas en la práctica
│   ├── diagramas   # Diagramas con las relaciones de los sistemas
│   └── memoria.odt # Archivo "en bruto" de la memoria entregada
├── memoria.pdf     # Archivo con la memoria entregada
├── README.md       # Este ficheor
├── servidor        # Código fuente del servicio "servidor"
├── servidor.bat    # Iniciador del servicio "servidor" (WIN)
├── servidor.sh     # Iniciador del servicio "servidor" (*nix)
├── usuario         # Código fuente del servicio "usuario"
├── usuario.bat     # Iniciador del servicio "usuario" (WIN)
└── usuario.sh      # Iniciador del servicio "usuario" (*nix)
```
